About SLib

This site developed for personal use as web development learning programming project.


Author: Santa Baltmane

SLib (short for "Santas Library").

To build this project used: Python programming language (Flask - web framework for Python), HTML, CSS, SQLAlchemy (database) and JavaScript for additional functuality.

Main features and Information about SLib: Collection of Artworks(Novels/Books, Movies, TV Shows and Animations) with wide range of data filtering options.

Image files are converted and saved into database as Binary data.

Additional features: Comment section under each Artwork, Options to create/add new Artwork and edit it any time, add additional tag/hashtag to each Artwork, delete/remove Artwork, brand as one of Favourite Artworks by adding star, rating each Artwork, change Artwork status as seen, planning to watch, watching now, default pictures for each Artwork Category if there is not image file added.

Data filtering by category, year, rate, status, tags and favourite.
